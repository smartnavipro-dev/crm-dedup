営業やマーケをやっていると、こういう話よく出てきませんか？

「SalesforceやHubSpotに同じ会社が2〜3件ずつ入っている」
「『株式会社山田商事』と『山田商事株式会社』が別レコードになってる」
「担当が変わるたびに、同じ取引先がまた新規登録されていく」

放置するほど積み上がるんですよね、これ。メールが二重送信される、営業履歴がバラバラで引き継げない、MAや分析ツールに連携すると数字がズレる。地味に、じわじわ効いてくる問題です。

「重要なのはわかってるけど、手作業で直すのは現実的じゃない」

そう感じている方のために、**CSVをアップロードするだけで重複を自動検出・整理できる無料ツール**を作りました。

🔍 **CRM重複データ整理ツール**
https://crm-dedup.streamlit.app/

---

### こんなことができます

**語順違い・法人格の違いも検出**
「株式会社山田商事」と「山田商事株式会社」のような語順・法人格の違いも同一企業として検出します。株式会社・有限会社・合同会社などを除去してから比較するので、よくある表記ゆれを見逃しません。

**電話番号・メールアドレスでも検出**
社名が違っていても、電話番号やメールが一致していれば重複候補として上がります。「03-1234-5678」と「0312345678」のようにハイフンの有無も自動で吸収します。

**削除前に1件ずつ自分で確認**
自動削除はしません。検出されたグループを画面に並べて表示するので、「どちらを残すか」を自分で判断してからエクスポートできます。誤検知はその場でスキップできます。

**出力は2種類**
- `cleaned.csv`：重複除去済みのデータ（元のCSVと同じ列構成）
- `deleted_records.csv`：削除対象の一覧（CRMでの照合・確認用）

**データはセッション終了後に消えます**
アップロードしたCSVはブラウザのセッション内だけで処理されます。顧客情報をサーバーに保持・記録しないので安心して使えます。
※AIオプション（任意）を使う場合のみ、AnthropicのAPIにデータが送信されます。

---

### 使い方はたったの5ステップ

① CRMからCSVをエクスポート（UTF-8・Shift-JIS対応、最大5,000行）
② ツールにアップロード
③ 重複を調べる列と検出感度を設定
④ 自動解析 → 検出グループをレビュー
⑤ 整理済みCSVをダウンロードしてCRMにインポート

登録不要、インストール不要。ブラウザからすぐ使えます。

---

### こんな方に使ってほしいです

- CRMのデータ整理を「いつかやらなきゃ」と後回しにしている
- 重複があるのはわかってるけど、件数が多くて手が出せない
- MAや分析ツールにCRMデータを連携する前にきれいにしたい
- CRMを乗り換える前にデータをクリーニングしておきたい

# ファクトチェックメモ（投稿前に確認済み）

| 記載内容 | コード上の根拠 |
|---|---|
| 最大5,000行 | `MAX_ROWS = 5_000` |
| UTF-8・Shift-JIS対応 | `read_csv_robust`でutf-8-sig/utf-8/shift-jis/cp932を順に試行 |
| 語順違い・法人格の違いを検出 | `normalize_name()`で株式会社等を除去してから`fuzz.ratio`で比較 |
| 対応法人格 | 株式会社・有限会社・合同会社・一般社団法人・医療法人ほか計16種 |
| 電話番号はハイフンの有無を吸収 | `normalize_phone()`で`re.sub(r'\D', '')`により数字のみ抽出して比較 |
| 自動削除しない | Step4でレビュー後、Step5でCSVエクスポートのみ |
| cleaned.csv / deleted_records.csv | Step5の`st.download_button`で確認済み |
| データ保存なし | Streamlitのセッションステートのみ、永続化処理なし |
| AIオプション使用時はAnthropicAPIに送信 | `claude_verify_group()`内でAPIコール、サイドバーに明記済み |

CRMの重複データは、放置するほど積み上がります。メールが二重送信される、営業履歴がバラバラで引き継げない、MAツールや分析ツールに連携すると数字がズレる——地味に、じわじわ効いてくる問題です。

「重要なのはわかってるけど、手作業で直すのは現実的じゃない」

そう感じている方に向けて、**CSVをアップロードするだけで重複を自動検出・整理できる無料ツール**を作りました。

🔍 **CRM重複データ整理ツール**
https://crm-dedup.streamlit.app/

---

### 主な機能

**語順違い・法人格の違いも検出**
「株式会社山田商事」と「山田商事株式会社」のように、語順や法人格の位置が違っていても同一企業として検出します。単純な文字列一致ではなく、株式会社・有限会社・合同会社などの法人格を取り除いてから比較しているためです。

**電話番号・メールアドレスでの検出**
社名が違っていても、電話番号やメールアドレスが一致していれば重複候補として検出できます。電話番号は「03-1234-5678」と「0312345678」のようにハイフンの有無を自動で吸収して比較します。

**削除前に1件ずつ確認**
自動削除は行いません。検出された重複グループを1件ずつ並べて表示し、「どちらを残すか」を人間が判断してからエクスポートします。誤検知はその場でスキップできます。

**2種類のCSVで出力**
- `cleaned.csv`：重複除去済みの最終データ（元のCSVと同じ列構成）
- `deleted_records.csv`：削除対象レコードの一覧（CRMでの照合・確認用）

**セッション終了後にデータは消去されます**
アップロードしたCSVはブラウザのセッション内のみで処理されます。顧客情報をサーバーに保持・記録しません。
※AIオプション（任意）を使用する場合のみ、AnthropicのAPIにデータが送信されます。

---

### 使い方（5ステップ）

① CRMからCSVをエクスポート（UTF-8またはShift-JIS対応、最大5,000行）
② ツールにアップロード
③ 重複を調べる列（会社名など）と検出感度を設定
④ 自動解析 → 検出されたグループをレビュー
⑤ 整理済みCSVをダウンロードしてCRMにインポート

登録不要、インストール不要。ブラウザからすぐ使えます。

---

### こういう方に使っていただいています

- CRMのデータ整理を後回しにし続けている
- 重複があることはわかっているが、件数が多くて手が出せない
- MAツールや分析ツールにCRMデータを連携する前にきれいにしたい
- CRM移行・リプレイスの前にデータをクリーニングしたい

# ファクトチェックメモ（投稿前に確認済み）

| 記載内容 | コード上の根拠 |
|---|---|
| 最大5,000行 | `MAX_ROWS = 5_000` |
| UTF-8・Shift-JIS対応 | `read_csv_robust`でutf-8-sig/utf-8/shift-jis/cp932を順に試行 |
| 語順違い・法人格の違いを検出 | `normalize_name()`で株式会社等を除去してから`fuzz.ratio`で比較 |
| 対応法人格 | 株式会社・有限会社・合同会社・一般社団法人・医療法人ほか計16種 |
| 電話番号はハイフンの有無を吸収 | `normalize_phone()`で`re.sub(r'\D', '')`により数字のみ抽出して比較 |
| 自動削除しない | Step4でレビュー後、Step5でCSVエクスポートのみ |
| cleaned.csv / deleted_records.csv | Step5の`st.download_button`で確認済み |
| データ保存なし | Streamlitのセッションステートのみ、永続化処理なし |
| AIオプション使用時はAnthropicAPIに送信 | `claude_verify_group()`内でAPIコール、サイドバーに明記済み |
